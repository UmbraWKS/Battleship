# Battleship
  Il gioco battaglia navale che ho fatto durante il terzo anno di scuola superiore mentre studiavo C.

## Dipendenze
- gcc
- make

## Installazione
  Come installare, compilare ed eseguire:
  ```
  git clone https://github.com/UmbraWKS/battleship.git
  cd battleship
  make
  ```

  Opzionale: `rm -rf .git`

  Esegui: `./battleship`
