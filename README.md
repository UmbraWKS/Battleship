# Chess
  The battleship game I made in my 3rd year of high school.

## Dependencies
- gcc
- make

## Installing
  Steps to install, compile and run
  ```
  git clone https://github.com/UmbraWKS/Battleship.git
  cd battleship
  make
  ```

  Optional: `rm -rf .git`

  Run with: `./battleship`
